hexo的基础使用方法

创建一个新的文章

git命令

1. 在hexo的文件目录下右键打开Git Bash Here

2. 创建文章

hexo new “标题”//创建一个新的文章

3. 命令行

hexo c //清除缓存
hexo g //本地更新文章
hexo d //上传至GitHub，更新网页

//注意当你挂复杂的梯子时候有时会发生hexo d失败，可以执行如下方式解决：
git config --global --unset http.proxy//清除代理地址

4. 完成
